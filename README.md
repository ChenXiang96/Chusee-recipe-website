## 项目简介
+ ### 1.简介</br>
“厨色”是一款以美食推荐和社交分享为核心结合数据挖掘技术</br>
从而实现菜品菜谱推送和用户动态实时分享的大型互联网站。</br>
+ ### 2.目标</br>
本网站致力打造：全面宣传、展示健康美食、美食食谱，发布美食信息资讯和</br>
广大美食爱好者建立良好互动的平台。在Internet普及的今天，利用互联网的</br>
优势，给美食爱好者与各地美食文化之间建立一座迅捷无伦的美好桥梁，无论对于</br>
美食文化传承还是便利大众生活都有十分重要的意义。

+ ### 3. 特色功能</br>
+ 1.动态发布</br>
用户登陆后，进入话题页，</br>
在页面左侧动态全部显示，</br>
右侧用户可实时发布话题</br>

+ 2.健康膳食</br>
用户进入健康页面可浏览</br>
网站筛选出的健康专题内容</br>
自主选择健康专题进行浏览学习</br>

+ 3.地图定位</br>
用户进入后可跳转到周边页，</br>
用户可以利用站内地图搜索</br>
自己感兴趣的城市和美食，</br>
位置将会在页内地图自动显示</br>

+ 4.推荐系统</br>
用户可登录推荐系统</br>
进行推荐内容的浏览</br>
还可对每个店铺进行</br>
评分评分越高，在首页</br>
的显示位置将越靠前</br>
